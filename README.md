# Navin D - Personal Portfolio v2

Welcome to the repository for my personal developer portfolio! This portfolio showcases my work as a Full Stack Developer, highlighting my projects, skills, experience, and certifications.

## 🔗 Live Site
[View my Live Portfolio here!](https://navin-d-portfolio.netlify.app/)

## ✨ Features
- **Dynamic Glassmorphism UI:** A sleek, modern aesthetic using glassmorphic cards, dynamic shadows, and vibrant, continuous gradient animations.
- **Scroll Animations:** Integrated with AOS (Animate On Scroll) for smooth, staggered reveal animations as you scroll down the page.
- **Seamless Document Viewing:** View PDF resumes and certificates directly on the page in a custom modal popup without breaking your flow.
- **Fully Responsive:** Custom flexbox and grid layouts optimized perfectly for mobile, tablet, and desktop viewing.

## 🛠️ Built With
- **HTML5** (Semantic structure)
- **CSS3** (Vanilla, CSS Variables, Keyframe Animations)
- **JavaScript** (Vanilla DOM Manipulation)
- **FontAwesome** (Icons)
- **AOS Library** (Scroll Animations)
